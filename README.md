Bienvenue sur **Memory**, un jeu interactif conçu pour tester la mémoire visuelle de manière ludique !  
Ce projet web est une excellente démonstration de compétences en développement Front-End avec HTML, CSS et JavaScript.

🔗 **Lien vers le jeu** : [Jouer maintenant](https://haja12233.github.io/Memory.github.io/)

## 🎯 Objectif du Projet

Le projet a plusieurs objectifs :

- 🎮 **Offrir une expérience de jeu simple et divertissante**, accessible à tous depuis un navigateur.
- 🧑‍💻 **Montrer mes compétences techniques** en JavaScript, gestion des événements, logique de jeu, et animations DOM.
- 🧠 **Stimuler la concentration et la mémoire** des utilisateurs tout en s’amusant.

---

## 🧠 Concept du Jeu

Le principe est simple et universel :

1. Une **grille de cartes** (faces cachées) est affichée.
2. Le joueur retourne **deux cartes** à la fois.
3. Si les deux cartes sont identiques, elles restent visibles.
4. Sinon, elles se retournent après un court délai.
5. Le but est de **trouver toutes les paires** avec le moins de tentatives possible.

---

## 🎮 Fonctionnement du Site

- À l’ouverture du jeu, une grille de cartes mélangées est générée.
- Chaque clic sur une carte déclenche :
  - Un **retournement visuel** (animation CSS/JS)
  - Une vérification de correspondance avec la carte précédemment retournée
- Les paires trouvées restent affichées.
- Le jeu se termine lorsque toutes les paires sont découvertes.
- Il est possible de **relancer une partie** à tout moment (ajout possible d’un bouton “Rejouer”).
